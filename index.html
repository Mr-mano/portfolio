DEVELOPPEUR WEB

Rennes - Saint Malo - Fougères - Dinan

Je m'appel Mano, j'habite en Bretagne, entre Rennes et Saint Malo. Passionné par le code, j'ai suivi une formation à Digital Campus Rennes, où j'ai obtenu une licence en développement web multimédia. Depuis je pratique, je m'intéresse, je progresse... Mon objectif : offrir le meilleur de la technologies allier à l'exigence d'une interface esthétique et fonctionnelle. Bref pour faire simple être bon dans mon domaine, tant par la qualité du code que par l'esthètisme du projet web que l'on me confit.

PHP

Php est le language server le plus utilisé pour communiquer avec la base de données.

Bootstrap

Bootstrap est un framework (html, css, javascript) utiles à la création du design. Il est très populaire

HTML

HyperText Markup Language est un language de balisage pour concevoir la structure du projet web

CSS

Cascading Style Sheets est un language permattant la mise en forme d'un projet écrit en html

Php MyAdmin

Généralement associé au logiciel MySqlWorkBench. Permet la créations de base de données sur mesure, en fonction des projets.

Sass

Sass est un préprocesseur CSS, qui permet de compiler le code est ainsi améliorer la vitesse de chargement.

Symfony

Symfony est un framework php puissant qui permet de faciliter le développement d'un site. Il est très utilisé en France et à travers le monde.

Wordpress

Wordpress est le CMS (Content Manager System) le plus utilisé au monde. Il est polyvalent et permet de développer toutes sorte de site.
